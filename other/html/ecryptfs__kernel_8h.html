ecryptfs_init_persistent_file : Fully initialized eCryptfs dentry object, with the lower dentry and the lower mount set

eCryptfs only ever keeps a single open file for every lower inode. All I/O operations to the lower inode occur through that file. When the first eCryptfs dentry that interposes with the first lower dentry for that inode is created, this function creates the persistent file struct and associates it with the eCryptfs inode.
