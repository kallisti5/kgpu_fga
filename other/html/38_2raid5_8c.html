static size_t scribble_len ( int  num)
static

scribble_len - return the required size of the scribble region - total number of disks in the array

The size must be enough to contain: 1/ a struct page pointer for each device in the array +2 2/ room to convert each entry in (1) to its corresponding dma (dma_map_page()) or page (page_address()) address.

Note: the +2 is for the destination buffers of the ddf/raid6 case where we calculate over all devices (not just the data blocks), using zeros in place of the P and Q blocks.
